Numeric cheat-sheet (full RGB mode)

  • Brightness (value): V ≈ (R+G+B)/3 0–60 very dark 60–110 dark 110–160 mid 160–210 light 210–255 very light
  • Muted vs vivid: S ≈ max−min 0–20 very muted 20–50 muted 50–90 moderate 90–170 vivid
  • Channel order: decide highest/lowest first (e.g., warm beige often R≥G≥B, cool slate often B≥G≥R).
  • Workflow: set value (move all channels together), set saturation (spread or compress channels), then fix hue (which channel leads).
